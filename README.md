# Figur

Financial lifestyle app landing page.

## Tech Stack

- **HTML5** - Semantic markup
- **Tailwind CSS v4** - Styling with custom theme
- **Inter Font** - Typography

## Color Palette
```css
Primary Blue: hsl(221, 100%, 50%)
Primary Navy: hsl(216, 94%, 21%)
Dark Blue: hsl(216, 71%, 14%)
Medium Gray: hsl(220, 9%, 46%)
Teal: hsl(161, 94%, 30%)
Mint: hsl(152, 81%, 96%)
```

## Features

- Fixed header with shadow
- Responsive hero section
- App store download CTAs
- Demo video integration

## Setup

1. Install dependencies
2. Build Tailwind: `npm run build:css`
3. Open `index.html`
